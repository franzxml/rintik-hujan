# Rintik Hujan

Halaman web lokal bertema hutan hujan neon biru dengan animasi tetesan hujan dan suara ambient.

## Cara Menjalankan
1. Letakkan semua file di satu folder bernama `Rintik-Hujan`.
2. Tambahkan file `hutan.png` (gambar latar belakang).
3. Tambahkan file `hujan.mp3` (suara hujan).
4. Buka `index.html` di browser.
